Key Features of HTML5 :

1. Semantic Elements

HTML5 introduces semantic elements like < header> , < nav> , < section> , and < footer> that provide better structure to web pages, making them more accessible and SEO-friendly.

2. Video and Audio Support

HTML5 includes native support for embedding video and audio using the < video> and < audio> elements, reducing the need for third-party plugins like Flash.

3. Local Storage

HTML5 offers the ability to store data locally in the user's browser using the localStorage and sessionStorage APIs, allowing for more efficient and responsive web applications.
